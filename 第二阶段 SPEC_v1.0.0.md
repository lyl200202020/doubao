## 1 引言

### 1.1 编写目的

本文档描述客户运营管理后台系统第二阶段（v0.0.2-v0.0.4）的软件需求规格，包括机构管理、用户管理、菜单管理、角色管理的功能需求、性能需求、安全需求等，为开发团队、测试团队提供统一的需求基准。

### 1.2 需求范围

**第二阶段包含以下版本**：
- **v0.0.2**：机构管理
- **v0.0.3**：用户管理 + 菜单管理
- **v0.0.4**：角色管理

### 1.3 术语定义

| 术语 | 定义 |
|------|------|
| 机构 | 组织架构的基本单元，采用树形结构管理 |
| 用户 | 系统操作主体，必须属于一个机构和一个角色 |
| 角色 | 控制用户可见菜单的功能权限集合 |
| 菜单 | 系统功能入口，分为目录、菜单、按钮三种类型 |
| 授权级别 | 用户属性，1-16 级，用于授权审批流程 |
| 逻辑删除 | 数据标记为删除状态，实际不从数据库删除 |

## 3.5 权限控制（v0.0.2-v0.0.4）

### 3.5.1 功能描述

权限控制实现基于角色的访问控制（RBAC），确保不同角色的用户登录后看到的菜单、页面、按钮不同。

**权限控制范围**：
1. **菜单可见性**：用户只能看到其角色关联的菜单
2. **路由访问控制**：用户只能访问其角色允许的页面路由
3. **按钮权限控制**：用户只能看到和点击其角色允许的按钮
4. **接口访问控制**：后端校验用户是否有权限调用接口

**注意**：权限控制 ≠ 授权管理
- **权限控制**：控制用户能看到什么菜单、能访问什么页面、能点击什么按钮（第二阶段）
- **授权管理**：控制业务审批流程，基于授权级别的审批权限（第三阶段）

### 3.5.2 功能清单

| 功能编号 | 功能名称 | 优先级 | 说明 |
|----------|----------|--------|------|
| F-PERM-001 | 动态菜单加载 | P0 | 根据用户角色动态生成左侧导航菜单 |
| F-PERM-002 | 路由访问控制 | P0 | 未授权路由自动拦截并重定向 |
| F-PERM-003 | 按钮权限控制 | P0 | 无权限按钮自动隐藏或禁用 |
| F-PERM-004 | 接口权限校验 | P0 | 后端校验接口调用权限 |

### 3.5.3 业务规则

| 规则编号 | 规则描述 | 验证方式 |
|----------|----------|----------|
| BR-PERM-001 | 用户登录时查询其角色关联的所有菜单 | 登录接口实现 |
| BR-PERM-002 | 菜单树根据用户角色过滤后返回 | 后端菜单树接口 |
| BR-PERM-003 | 前端根据菜单树动态生成路由 | 路由守卫实现 |
| BR-PERM-004 | 访问未授权路由时重定向到 403 页面 | 路由守卫实现 |
| BR-PERM-005 | 按钮使用 v-permission 指令控制显示/隐藏 | 前端指令实现 |
| BR-PERM-006 | 后端接口使用@RequirePermission 注解校验权限 | 后端注解实现 |
| BR-PERM-007 | 无权限调用接口时返回 403 状态码 | 后端拦截器实现 |

### 3.5.4 权限模型

```
用户（User） → 角色（Role） → 菜单（Menu）
  ↓
可访问的菜单集合
  ↓
动态生成路由 + 按钮权限控制
```

### 3.5.5 权限标识（perms）

**权限标识格式**：`模块：操作：资源`

| 权限标识示例 | 说明 |
|--------------|------|
| `sys:org:view` | 查看机构 |
| `sys:org:add` | 新增机构 |
| `sys:org:edit` | 编辑机构 |
| `sys:org:delete` | 删除机构 |
| `sys:user:view` | 查看用户 |
| `sys:user:add` | 新增用户 |
| `sys:user:edit` | 编辑用户 |
| `sys:user:delete` | 删除用户 |
| `sys:menu:view` | 查看菜单 |
| `sys:menu:add` | 新增菜单 |
| `sys:menu:edit` | 编辑菜单 |
| `sys:menu:delete` | 删除菜单 |
| `sys:role:view` | 查看角色 |
| `sys:role:add` | 新增角色 |
| `sys:role:edit` | 编辑角色 |
| `sys:role:delete` | 删除角色 |

### 3.5.6 数据字典

**权限类型（perm_type）**：

| 值 | 含义 | 说明 |
|----|------|------|
| 1 | 菜单权限 | 控制菜单可见性 |
