## 三、功能实现详情

### 3.1 后端实现

#### 3.1.1 项目结构（DDD四层架构）

```
customer-admin-platform/
├── platform-biz/                    # 业务实现
│   └── src/main/java/.../platform/
│       └── sys/                     # 系统管理领域
│           ├── domain/              # 领域层
│           │   ├── entity/          # 实体（SysUser, SysRole, SysMenu, SysOrg）
│           │   ├── repository/      # 仓储接口
│           │   └── service/         # 领域服务（AuthDomainService）
│           ├── application/         # 应用层
│           │   ├── command/         # 命令对象（LoginCommand）
│           │   └── service/         # 应用服务（AuthApplicationService）
│           ├── infrastructure/      # 基础设施层
│           │   ├── dao/             # DAO接口
│           │   └── repository/      # 仓储实现
│           └── api/                 # 接口层
│               ├── controller/      # 控制器（AuthController）
│               └── vo/              # 视图对象（LoginVO）
│
└── platform-start/                  # 启动模块
    └── src/main/java/.../
        ├── PlatformApplication.java # 启动类
        └── config/                  # 配置类
```

#### 3.1.2 已实现接口

| 接口 | 方法 | 路径 | 说明 | 状态 |
|------|------|------|------|------|
| 登录 | POST | /api/auth/login | 用户名密码登录 | ✅ |
| 退出 | POST | /api/auth/logout | 退出登录 | ✅ |
| 刷新Token | POST | /api/auth/refresh | 刷新Token | ✅ |

#### 3.1.3 核心组件

| 组件 | 文件 | 说明 |
|------|------|------|
| SM3Util | common-core/util/SM3Util.java | 国密SM3哈希算法 |
| SM4Util | common-core/util/SM4Util.java | 国密SM4对称加密 |
| JwtUtil | common-security/jwt/JwtUtil.java | JWT Token生成与解析 |
| SecurityConfig | common-security/config/SecurityConfig.java | Spring Security配置 |
| JwtAuthenticationFilter | common-security/filter/JwtAuthenticationFilter.java | JWT认证过滤器 |
| Result | common-core/result/Result.java | 统一返回封装 |
| ErrorCode | common-core/enums/ErrorCode.java | 错误码枚举 |
| BusinessException | common-core/exception/BusinessException.java | 业务异常 |
