## 三、关键技术实现

### 3.1 密码加密

使用 SM3 国密算法加密用户密码：
```java
String encryptedPassword = SM3Util.hash(password);
```

### 3.2 逻辑删除

所有实体类添加 `@TableLogic` 注解：
```java
@TableLogic
private Integer deleted;
```

### 3.3 自动填充

使用 MyBatis-Plus 自动填充字段：
```java
@TableField(fill = FieldFill.INSERT)
private LocalDateTime createdTime;

@TableField(fill = FieldFill.INSERT_UPDATE)
private LocalDateTime updatedTime;
```

### 3.4 树形结构构建

递归构建树形结构：
```java
private List<OrgVO> buildOrgTree(List<SysOrg> allOrgs, String parentId) {
    return allOrgs.stream()
        .filter(org -> parentId == null ? org.getParentOrgId() == null : parentId.equals(org.getParentOrgId()))
        .map(org -> {
            OrgVO vo = convertToVO(org);
            vo.setChildren(buildOrgTree(allOrgs, org.getOrgId()));
            return vo;
        })
        .collect(Collectors.toList());
}
```

## 七、注意事项

1. **数据库字段命名**：所有表使用下划线命名法，实体类使用驼峰命名法
2. **逻辑删除**：所有查询需要添加 `deleted = 0` 条件
3. **权限标识**：按钮级别权限使用 `perms` 字段，格式为 `sys:module:action`
4. **树形查询**：机构树和菜单树使用递归构建
5. **密码安全**：用户密码必须使用 SM3 加密
6. **事务管理**：所有写操作添加 `@Transactional` 注解
7. **参数校验**：Controller 层使用 `@Valid` 注解校验请求参数
