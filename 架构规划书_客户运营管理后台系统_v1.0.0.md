## 2 项目结构

### 2.1 技术架构说明

采用**运营平台 + 业务服务分离架构**：
- **运营平台（admin-platform）**：系统管理、授权管理、运营配置
- **业务服务（crm-service等）**：独立业务逻辑，通过Dubbo RPC调用
- 公共模块统一管理，各服务依赖

## 5 国密算法集成方案

### 5.1 国密算法清单

| 算法 | 用途 | 说明 |
|------|------|------|
| SM3 | 密码加密 | 用于用户密码摘要存储 |
| SM4 | 对称加密 | 用于敏感数据加密存储 |
| SM2 | 非对称加密 | 用于数字签名 |

### 5.2 密码存储方案

```java
@Component
public class Sm3Util {
    
    public static String encrypt(String password) {
        byte[] data = password.getBytes(StandardCharsets.UTF_8);
        byte[] hash = DigestUtil.sm3(data);
        return HexUtil.encodeHexStr(hash);
    }
    
    public static boolean verify(String rawPassword, String encryptedPassword) {
        return encrypt(rawPassword).equalsIgnoreCase(encryptedPassword);
    }
}
```
