## 四、运行指南

### 4.1 安装依赖

```bash
cd customer-admin-web
npm install
```

### 4.2 启动开发服务器

```bash
npm run dev
```

项目将在 http://localhost:3000 启动

### 4.3 构建生产版本

```bash
npm run build
```

## 五、接口配置

### 5.1 后端接口地址

| 环境 | 地址 |
|------|------|
| 开发环境 | http://localhost:8080 |
| 代理配置 | vite.config.ts |

### 5.2 接口列表

| 序号 | 接口名称 | 方法 | 路径 |
|------|----------|------|------|
| 1 | 登录 | POST | /api/auth/login |
| 2 | 退出 | POST | /api/auth/logout |
| 3 | 获取当前用户 | GET | /api/auth/current |
| 4 | 获取用户信息 | GET | /api/sys/user/info |
| 5 | 获取菜单树 | GET | /api/sys/menu/tree |

## 六、与后端对接说明

### 6.1 接口返回格式

统一响应格式：
```json
{
  "code": 200,
  "msg": "操作成功",
  "data": {},
  "timestamp": 1706740000000
}
```

### 6.2 错误处理

| 状态码 | 处理方式 |
|--------|----------|
| 200 | 成功，返回data |
| 401 | 跳转登录页 |
| 403 | 提示无权限 |
| 500 | 提示服务器错误 |
