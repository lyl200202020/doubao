## 常见问题排查

### 问题 1：服务启动失败

**检查项**：
1. 端口 8080 是否被占用
2. 数据库连接配置是否正确
3. Redis 连接配置是否正确

**解决方法**：
```bash
# 查看端口占用
netstat -ano | findstr :8080

# 修改端口（如果被占用）
编辑 application-local.yml，修改 server.port
```

### 问题 2：数据库连接失败

**检查项**：
1. MySQL 服务是否启动
2. 数据库用户名密码是否正确
3. 数据库是否存在

**解决方法**：
```bash
# 测试数据库连接
mysql -h 113.221.57.26 -u root -p

# 创建数据库
CREATE DATABASE cust DEFAULT CHARSET=utf8mb4;
```

### 问题 3：Redis 连接失败

**检查项**：
1. Redis 服务是否启动
2. Redis 密码是否正确

**解决方法**：
```bash
# 测试 Redis 连接
redis-cli -h 113.221.57.26 -p 6379 -a mm123456

# 查看 Redis 服务状态
```
